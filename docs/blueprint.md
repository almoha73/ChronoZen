# **App Name**: ChronoZen

## Core Features:

- Countdown Display: Display a digital-style countdown timer on the main screen.
- Preset Timers: Provide quick-selection buttons for preset durations: 3:30, 5:00, 10:00, 15:00, 20:00, 25:00, and 30:00.
- Timer Control: Implement start/pause/reset functionality with a single button that toggles between states.
- Progress Animation: Incorporate a progress animation (circular ring or dynamic bar) to visually represent the time remaining.
- Vibration Alert: Use the device's vibration motor at the end of the countdown to alert the user.
- Smart Animation Pacing: AI tool that adjusts the progress animation speed based on the selected time and the remaining time to keep the user informed of progress. 
- Theme Toggle: Implement an automatic light/dark mode based on the system settings.

## Style Guidelines:

- Primary color: Saturated blue (#29ABE2) to evoke a sense of calm and focus.
- Background color: Light gray (#F0F4F8) for a clean and modern feel.
- Accent color: Warm orange (#FF9933) to highlight interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look, suitable for both headlines and body text.
- Use simple, minimalist icons for start/pause/reset actions.
- Ensure a responsive design, primarily optimized for mobile devices with tablet compatibility.
- Incorporate subtle animations for transitions, such as a fade-in effect on launch and a zoom effect on button clicks.
